instantiation: creates a new object of the Calculator class in memory.
              Make me a new calculator object and store it in a variable called calc
              Think of Calculator as a blueprint
              new Calculator() as building a real calculator based on that blueprint.
              The variable calc is the label on that specific calculator.

instance method : public void add(double val) { value = value + val; }
public : Access modifier — means the method can be called from outside the class.
void: Return type — this method doesn’t return anything (it just performs an action).
add: Method name — what you call when using it (e.g. calc.add(5.0)).
(double val): Parameter list — this method expects one argument of type double.
{ value = value + val; }: Method body — the statements that define what the method does.
